| &#10003; | **Particle Background** | Interactive particle network on the home page |

## &#9881; Getting Started

```bash
# Clone the repository
git clone https://github.com/MuhammadOkasha004/MyPortfolio.git

# Navigate to the project
cd MyPortfolio

# Open in browser
start index.html
```

> No build tools or package managers required &mdash; open `index.html` in any modern browser.
