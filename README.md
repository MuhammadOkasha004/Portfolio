## ✦ Overview

A modern, responsive **developer portfolio** with a dark/light theme, particle animations, and a cyberpunk-inspired aesthetic — built entirely with vanilla HTML, CSS & JavaScript. No frameworks. No build tools. Just open and go.

---

## ✦ Pages

| Page | Description |
|------|-------------|
| 🏠 **Home** | Hero section · Typing animation · Particle canvas · Stats badges |
| 👤 **About** | Bio · Terminal-style info card · Personal stats |
| 🛠 **Skills** | Categorized skill cards with progress indicators |
| 🚀 **Projects** | Project showcase with live demo & source links |
| 📬 **Contact** | Contact form · Social & professional links |

---

## ✦ Features

| Feature | Details |
|---------|---------|
| 🌗 **Dark / Light Theme** | Seamless toggle with smooth transitions |
| ✨ **Particle Background** | Interactive canvas network on the hero |
| ⌨️ **Typing Animation** | Dynamic text rotation in the hero section |
| 📱 **Responsive Design** | Optimized for mobile, tablet & desktop |
| 🎞 **Scroll Reveal** | Elements animate into view on scroll |
| 🖱 **Cursor Trail** | Accent-coloured cursor follower effect |
| 💡 **Holographic Hero** | SVG illustration with animated glow rings |
| 📩 **Contact Form** | Functional form with integration |

## ✦ Getting Started

```bash
# 1. Clone the repo
git clone https://github.com/MuhammadOkasha004/MyPortfolio.git

# 2. Enter the directory
cd MyPortfolio

# 3. Open in browser
start index.html   # Windows
open index.html    # macOS
```

> ✅ No build tools, no package managers — just open `index.html` in any modern browser.
